# 위대한 청춘

[Portfolio](https://lisakim522-lab.github.io/portfolio/)

## 작품 연혁

`위대한 청춘`은 2022년부터 2023년까지 이어진 영상스토리 가곡콘서트로, 세대의 기억과 가족애, 한국 근현대사의 시간을 영상과 노래로 함께 풀어낸 작품입니다.

2022년에는 충북 제천의림지수변무대를 시작으로 예술의전당 IBK챔버홀 무대까지 이어지며 작품의 주요 흐름이 형성되었습니다.

2023년에는 푸르지오아트홀(2회), 안성시 안성맞춤아트홀, 안성 LS미래원, 제주해비치아트페스티벌 쇼케이스, 경기아트센터 대극장까지 공연이 확장되며 작품의 무대와 관객층이 넓어졌습니다.
